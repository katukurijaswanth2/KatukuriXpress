
# 📦 KatukuriXpress – Modern React E-commerce App

KatukuriXpress is a fully responsive e-commerce web application built using modern React practices. It demonstrates scalable architecture, reusable components, and efficient state management while integrating external APIs for dynamic product data.

🔗 **Live Demo:** https://katukurixpress.vercel.app/  
🔗 **GitHub Repo:** https://github.com/katukurijaswanth2/KatukuriXpress  

---

## 🚀 Features

- 🛍️ Product listing with dynamic API data  
- 🔎 Category-based filtering system  
- 🛒 Cart management (add/remove/update quantity)  
- ⭐ Product rating system  
- ⏳ Skeleton loaders for better UX (instead of traditional spinners)  
- 🔐 Authentication UI (Login / Signup)  
- 🎯 Deals of the Day with countdown timer  
- 📱 Fully responsive design  

---

## 🧠 Tech Stack

- **Frontend:** React + Vite  
- **Routing:** React Router  
- **State Management:** Redux  
- **Styling:** CSS  
- **API Handling:** Fetch / Async Hooks  
- **Deployment:** Vercel

### Key Highlights:

- **Separation of Concerns**
  - `features/` → Business logic  
  - `shared/` → Reusable UI components  
  - `utilities/` → Helper logic  

- **Custom Hooks**
  - Encapsulated reusable logic for API calls and state handling  

- **Skeleton Loading Strategy**
  - Replaced traditional loaders with skeleton UI  

---

## ⚡ Advanced Concepts Used

- Props Destructuring  
- React Hooks (useState, useEffect, custom hooks)  
- Component Reusability  
- Lifting State Up  
- Conditional Rendering  
- Dynamic Rendering from APIs  
- Modular CSS Architecture  

---

## 🧩 Folder Structure Overview

- `features/` → Core modules (cart, products, deals, auth)  
- `shared/` → Navbar, Footer, reusable components  
- `utilities/` → Filtering logic  
- `mainLayout/` → Layout wrapper  
- `appRouterDom/` → Routing setup  
- `redux/` → Global state management  
- `assets/` → Static files

## 🛠️ Installation & Setup

```bash

# Navigate into project
cd KatukuriXpress

# Install dependencies
npm install

npm run dev

```
